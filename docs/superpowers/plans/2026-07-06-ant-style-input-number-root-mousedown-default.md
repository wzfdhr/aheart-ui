# InputNumber Root Mouse-Down Default Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `AInputNumber` root `mousedown` prevent default before invoking user root mouse-down listeners when the root, prefix, suffix, addon, or controls area is pressed.

**Architecture:** Keep the existing root/input attr split. Remove `onMousedown`/`onMouseDown` from generic `rootAttrs`, then invoke those handlers from the internal `handleRootMouseDown` after focus/default-prevention logic.

**Tech Stack:** Vue 3 `<script setup>`, TypeScript, Vitest with Vue Test Utils, VitePress docs.

### Task 2: Implement Internal Mouse-Down Listener Dispatch

**Files:**
- Modify: `packages/components/src/input-number/input-number.vue`

- [x] **Step 1: Add mouse-down attr helpers**

Add helpers near the root mouse attr helpers:

```ts
const rootMouseDownEventAttrs = new Set(['onMousedown', 'onMouseDown'])
const isRootMouseDownEventAttr = (key: string) => rootMouseDownEventAttrs.has(key)
const callMouseEventHandler = (handler: unknown, event: MouseEvent) => {
  if (Array.isArray(handler)) {
    handler.forEach((item) => callMouseEventHandler(item, event))
    return
  }

  if (typeof handler === 'function') {
    ;(handler as (event: MouseEvent) => void)(event)
  }
}
```

- [x] **Step 2: Exclude mouse-down from generic root attrs**

Update `rootAttrs` so it includes root mouse events except mouse-down:

```ts
if (isRootMouseEventAttr(key) && !isRootMouseDownEventAttr(key)) {
  result[key] = value
}
```

Leave `inputAttrs` excluding all root mouse events as it does now.

- [x] **Step 3: Prevent default and dispatch user mouse-down handlers**

Update `handleRootMouseDown` to:

```ts
const handleRootMouseDown = (event: MouseEvent) => {
  if (inputRef.value && event.target !== inputRef.value) {
    inputRef.value.focus()
    event.preventDefault()
  }

  rootMouseDownEventAttrs.forEach((key) => {
    callMouseEventHandler(attrs[key], event)
  })
}
```
