# InputNumber Spinbutton ARIA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add rc-compatible spinbutton ARIA metadata to the inner `AInputNumber` input.

**Architecture:** Keep the existing template bindings and value computation. Add lightweight computed ARIA values next to input attr helpers and bind them directly on the native `<input>`.

**Tech Stack:** Vue 3 `<script setup>`, TypeScript, Vitest with Vue Test Utils, VitePress docs.

### Task 1: Add RED Coverage

- [x] **Step 1: Write the failing test**

Add this test near the initial render/native attribute tests:

```ts
it('exposes spinbutton ARIA metadata on the inner input', () => {
  const wrapper = mount(InputNumber, {
    props: { modelValue: 4, min: 1, max: 10 }
  })
  const input = wrapper.find('input')

  expect(input.attributes('role')).toBe('spinbutton')
  expect(input.attributes('aria-valuemin')).toBe('1')
  expect(input.attributes('aria-valuemax')).toBe('10')
  expect(input.attributes('aria-valuenow')).toBe('4')
})
```

Add a boundary test for the empty string case:

```ts
it('omits aria-valuenow when the current value is empty', () => {
  const wrapper = mount(InputNumber, {
    props: { modelValue: '', stringMode: true }
  })

  expect(wrapper.find('input').attributes('aria-valuenow')).toBeUndefined()
})
```

### Task 2: Implement Spinbutton ARIA Bindings

**Files:**
- Modify: `packages/components/src/input-number/input-number.vue`

- [x] **Step 1: Add computed ARIA value**

Add this computed near `inputMode` and `inputAttrs`:

```ts
const ariaValueNow = computed(() => {
  if (mergedValue.value === undefined || mergedValue.value === '') {
    return undefined
  }

  return String(mergedValue.value)
})
```

- [x] **Step 2: Bind ARIA metadata on the input**

Add these bindings to the native `<input>`:

```vue
role="spinbutton"
:aria-valuemin="min"
:aria-valuemax="max"
:aria-valuenow="ariaValueNow"
```
