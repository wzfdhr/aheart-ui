# FormItem Message Variables Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `AFormItem` `messageVariables` interpolation for validation rule messages.

**Architecture:** `AFormItem` resolves field-level message variables and registers them with `AForm`. `AForm` stores those variables per field and interpolates `${key}` placeholders only when returning a failing rule message.

**Tech Stack:** Vue 3, TypeScript, Vitest, VitePress, pnpm.
