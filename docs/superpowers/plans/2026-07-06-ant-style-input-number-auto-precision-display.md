# InputNumber Auto Precision Display Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Infer default displayed decimal places from `step` when `precision` is not explicitly provided.

**Architecture:** Extend the existing default display formatter with an auto precision helper. Keep explicit `precision`, pending typed text, custom formatter behavior, and emitted values unchanged.

**Tech Stack:** Vue 3, TypeScript, Vitest, VitePress, existing aheart-ui component build.
