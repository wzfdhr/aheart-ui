# InputNumber Root Focus Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Focus the inner `AInputNumber` input when the outer root receives a mouse-down, matching rc InputNumber root focus behavior.

**Architecture:** Keep existing root/input attribute splitting. Add one internal root mouse-down handler in `input-number.vue`; Vue will merge it with existing root listener forwarding so user-provided root mouse listeners continue to run.

**Tech Stack:** Vue 3 `<script setup>`, TypeScript, Vitest with Vue Test Utils, VitePress docs.

### Task 2: Implement Root Mouse-Down Focus

**Files:**
- Modify: `packages/components/src/input-number/input-number.vue`

- [x] **Step 1: Add the internal handler**

Add this helper near the other root/input event helpers:

```ts
const handleRootMouseDown = (event: MouseEvent) => {
  if (inputRef.value && event.target !== inputRef.value) {
    inputRef.value.focus()
  }
}
```

- [x] **Step 2: Wire the root wrapper**

Update the root `<span>` to include:

```vue

```
