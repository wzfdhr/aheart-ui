# InputNumber Step Offset Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Align `AInputNumber` `step` event offset with Ant-style unsigned, original-step semantics.

**Architecture:** Keep `handleStep` as the shared event emission path. Pass a display/event offset separately from the signed internal direction and let `type` carry the direction.

**Tech Stack:** Vue 3, TypeScript, Vitest, VitePress, existing aheart-ui component build.
