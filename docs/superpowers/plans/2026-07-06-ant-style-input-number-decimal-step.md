# InputNumber Decimal Step Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prevent floating point artifacts during `AInputNumber` numeric decimal stepping.

**Architecture:** Route all step target calculation through the component's decimal string helpers, then convert back to number outside `stringMode`.

**Tech Stack:** Vue 3, TypeScript, Vitest, VitePress, existing aheart-ui component build.
